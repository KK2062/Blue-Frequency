# 🌊 Blue Frequency

**An interactive ocean health visualization using the science of blue.**

Blue Frequency maps real-world ocean data to a shifting spectrum of blue — healthy oceans glow deep navy and cobalt, while stressed regions fade to pale, washed-out gray-blue. The color itself tells the story.

## What it does

- Animates a layered ocean canvas for 5 ocean regions
- Displays real-world approximated data: temperature, pH, pollution index, and health score
- Shifts the blue color palette based on ocean health — the worse the health, the more the color degrades
- Explains what each shade of blue represents scientifically

## Ocean regions

| Region | Health Score |
|---|---|
| Deep Sea | 88% |
| Pacific Ocean | 72% |
| Atlantic Ocean | 54% |
| Arctic Ocean | 41% |
| Coral Triangle | 35% |

## Built with

- HTML5 Canvas API (animation)
- Vanilla JavaScript (no frameworks needed)
- CSS custom properties
- Google Fonts: Syne + DM Mono

## How to run

Just open `index.html` in any browser — no installs, no build steps.

Or view it live at: `https://yourusername.github.io/blue-frequency`
